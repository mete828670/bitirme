Sırayla şu komutları yazınız:

Mssql yükleyin:
sudo ./install_mssql.sh

ODBC ile ilgili gerekli paketleri yükleyin:
sudo ./odbc.sh

ÖNEMLİ: Python dosyaları içerisindeki sunucu ip adresi ve veritabanı şifresini güncelleyin.
python3 create_database.py
python3 ex_database_setup.py
python3 edit_database.py
